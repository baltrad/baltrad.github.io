Introduction

Welcome to BALTRAD! This page is the first place to come to become aquainted with us. We hope you will like what you see and read.

We have divided up this documentation into several parts. The FAQ (Frequently Asked Questions) should probably be consulted first, because it will help you understand what BALTRAD is, and the principles guiding it, whereas the User Guide is more focussed on helping you get started with the system. Another way of putting it is that the FAQ will give you an overview and the User Guide will give you more substance. You can also find detailed information about different modules in BALTRAD community-based weather radar networking. Information about the releases can be found here and if you are interested in algorithms you can find them here.

The finest level of detail will be available in each software package's documentation, which is available when it is built. If you build a so-called "BALTRAD node", the documentation of the packages you decide to build will be built automatically, and that is where to look for such details.

Enjoy!
